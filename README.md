# 烫画机声音计数器（手机端）

手机麦克风监听烫画机提示音，实时显示：

- 本次累计件数
- 最近 60 秒完成件数
- 预计每小时产能
- 平均每件耗时
- 本机自动保存记录
- 监听时保持屏幕常亮

## GitHub Pages

在仓库中打开 `Settings → Pages`：

- Source：`Deploy from a branch`
- Branch：`main`
- Folder：`/(root)`

发布地址：

`https://q306395528-wq.github.io/heatpress-counter/`

## 手机使用

1. 用 Safari 或 Chrome 打开发布地址。
2. 点击“开始监听”，允许麦克风权限。
3. 观察声音强度，调节声音阈值。
4. 工作时保持网页在前台，锁屏或切换 App 可能漏计。
5. 开始和结束各响一次的机器，选择“2 次声音 = 1 件”。

声音只在手机本地实时分析，不上传录音。
